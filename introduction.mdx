## Welcome to Atomix QRGen

Atomix QRGen is a modern, privacy-focused QR code generator built with Astro, Preact, and Tailwind CSS. Generate customizable QR codes for multiple formats including URLs, WiFi credentials, vCards, and plain text — all fully client-side. Support for calendar events and payment QR codes is coming soon.

**No tracking. No backend. No stored data.**

## Technology stack

Atomix QRGen leverages modern web technologies for optimal performance:

- **[Astro](https://astro.build)** - Static site generator with partial hydration for minimal JavaScript
- **[Preact](https://preactjs.com)** - Lightweight React alternative (3KB) for interactive components
- **[Tailwind CSS](https://tailwindcss.com)** - Utility-first CSS framework for responsive design
- **[qr-code-styling](https://github.com/kozakdenys/qr-code-styling)** - Customizable QR code generation library

## Why client-side?

Generating QR codes entirely in the browser provides several advantages:

<AccordionGroup>
  <Accordion title="Privacy & security">
    Your sensitive data (WiFi passwords, payment details, contact info) never leaves your device. No server storage means no data breaches or privacy concerns.
  </Accordion>
  
  <Accordion title="Zero latency">
    QR codes generate instantly without network roundtrips. Works perfectly offline once the page is loaded.
  </Accordion>
  
  <Accordion title="Cost-effective hosting">
    Static site deployment is cheap or free. No backend infrastructure, databases, or server maintenance required.
  </Accordion>
  
  <Accordion title="Full transparency">
    Open source code means you can verify exactly what happens with your data. No hidden tracking or analytics.
  </Accordion>
</AccordionGroup>
